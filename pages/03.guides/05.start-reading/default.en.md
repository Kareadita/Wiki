### Keyboard Shortcuts
Each reader shares the same keyboard shortcuts:
* B: Bookmark page (Comic/Manga reader)
* F: Toggle fullscreen mode
* Left/Right arrow: Move next/prev
* ESC: Close the reader
* G: Go to page
